# Doc Appoint
A modern full stack doctor appointment  booking platform where user can browse doctor and search and dynamically show doctor details and can book appointment securely  with authentication


## Live Features
* Authentication with Better Auth
* Google login 
* JWT Token protected routes
* Dynamic doctor archive page
* search doctor name/ Specialty
* Dynamic doctor details page
* Booking system 
* user dashboard
* my booking dynamic data
* delete booking  


#  Tech Stack

## Frontend

* Next.js 15
* React
* Tailwind CSS
* DaisyUI
* React Icons
* React Toastify

## Backend

* Express.js
* MongoDB
* JWT
* Better Auth
